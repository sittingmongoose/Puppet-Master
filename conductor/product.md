# Product Definition

## Initial Concept
RWM Puppet Master

## Goals
- Orchestrate multiple AI CLI platforms (Cursor, Codex, Claude Code, Gemini, Copilot).
- Provide a native desktop GUI for managing AI development workflows.
- Enforce a structured, four-tier hierarchical workflow (Phase -> Task -> Subtask -> Iteration).

## Target Audience
- Inexperienced developers looking to Vibecode.  Using the subscriptions from Gemini, Cursor, Claude Code, Github Copilot, Codex.

## Key Features
- Cross-platform (Windows, macOS, Linux) with native installers.
- Four-tier workflow: Phase → Task → Subtask → Iteration.
- File-based memory (`progress.txt`, `AGENTS.md`) and state management.
- Integration with 5 specific AI CLI platforms (Cursor, Codex, Claude Code, Gemini, Copilot).
- Utilizes Ralph Loops.
- Comprehensive planning and interview phase at the start to identify and fill knowledge gaps.

## User Experience
- A native desktop GUI built with Rust and Iced.
- Automated orchestration with minimal user intervention.
