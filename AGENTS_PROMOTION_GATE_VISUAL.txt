═══════════════════════════════════════════════════════════════════
                         KEY METRICS
═══════════════════════════════════════════════════════════════════

Usage Score:
  = min(count, 10) / 10
  Examples:
    • 3 uses  → 0.30
    • 5 uses  → 0.50
    • 10 uses → 1.00
    • 15 uses → 1.00 (capped)

Success Score:
  = successes / count
  Examples:
    • 10/10 → 1.00
    • 8/10  → 0.80
    • 5/10  → 0.50
    • 2/10  → 0.20

Promotion Score:
  = (usage_score + success_score) / 2
  Examples:
    • 10 uses, 10/10 success → (1.0 + 1.0) / 2 = 1.00 ✓ PROMOTE
    • 5 uses, 5/5 success    → (0.5 + 1.0) / 2 = 0.75 ✗ TOO LOW
    • 10 uses, 7/10 success  → (1.0 + 0.7) / 2 = 0.85 ✓ PROMOTE
    • 3 uses, 2/3 success    → (0.3 + 0.67) / 2 = 0.48 ✗ TOO LOW
